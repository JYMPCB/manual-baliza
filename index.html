1 Conexión inicial

La baliza crea su propia red Wi-Fi para que pueda ser controlada desde un celular, tablet o computadora. No es necesario instalar ninguna aplicación.

1

Encienda la baliza

Espere unos segundos hasta que el equipo complete el encendido.

2

Conéctese al Wi-Fi

Desde el celular, active Wi-Fi y seleccione la red de la baliza.

3

Abra el navegador

Ingrese esta dirección directamente en la barra superior:

http://192.168.4.1/
4

Controle la baliza

Se abrirá la aplicación web con todas las funciones disponibles.

Importante: el teléfono puede mostrar que la red “no tiene Internet”. Es normal: esa red se utiliza únicamente para comunicarse con la baliza. Mantenga la conexión activa.

2 Pantalla de inicio

La pantalla de inicio permite consultar el estado de la batería, regular el brillo y seleccionar una secuencia rápida.

Estado de batería

  • Voltaje: tensión actual de la batería.
  • Corriente: consumo instantáneo del equipo.
  • Potencia: energía utilizada en ese momento.
  • Carga: porcentaje aproximado de batería disponible.
  • Tiempo restante: estimación de autonomía según el uso actual.

Control de brillo

Use la barra de brillo para ajustar la intensidad general de los LEDs.

  • Menor brillo: mayor autonomía.
  • Mayor brillo: mejor visibilidad con luz ambiente intensa.
  • Para uso nocturno se recomienda un brillo medio o bajo.

3 Secuencia rápida

La sección Secuencia rápida permite cambiar la señalización en pocos segundos. Seleccione un modo, elija un perfil de color y presione Aplicar.

Inicial

Modo de arranque predeterminado. Ejecuta tres secuencias genéricas de destello para comenzar a utilizar la baliza inmediatamente.

Eco

Modo de bajo consumo. Utiliza una secuencia optimizada para maximizar la autonomía de la batería.

Full

Modo de alta luminosidad. Ofrece la mayor intensidad de señalización, con un consumo de batería superior.

Perfiles de color

Seleccione el perfil que mejor se adapte a la señalización requerida:

  • Bomberos
  • Policía
  • Vialidad
  • Ambulancia

Aplicar configuración

Luego de seleccionar el modo y el perfil de color, presione:

▶ Aplicar

La baliza cambiará a la configuración elegida.

4 Configuración Wi-Fi

Esta sección permite conectar la baliza a una red Wi-Fi existente. Es una función opcional: el equipo puede utilizarse directamente desde su propia red Wi-Fi.

  1. Ingrese a la pestaña Configuración.
  2. Presione Escanear redes.
  3. Seleccione la red Wi-Fi disponible.
  4. Ingrese la contraseña correspondiente.
  5. Espere la confirmación de conexión.

Desconectar

Corta la conexión actual de la baliza a la red seleccionada. Úselo cuando quiera cambiar de red o volver al funcionamiento local.

Olvidar red

Elimina del equipo los datos guardados de esa red Wi-Fi. Es útil si cambió la contraseña o si la baliza será utilizada en otro lugar.

Cuando la conexión sea correcta, la pantalla mostrará el estado: Conectado a [nombre de red].

5 Secuencia avanzada

El modo avanzado permite crear una secuencia personalizada de destellos, colores y movimientos. La baliza ejecutará los pasos en el orden configurado.

Campos de cada paso

  • Patrón: tipo de efecto luminoso.
  • Velocidad: rapidez de la animación.
  • Duración: tiempo de reproducción del paso.
  • Color: color utilizado por el efecto.

Patrones disponibles

  • Recorrido (chase): el efecto se desplaza alrededor de la baliza.
  • Expansión desde lados: la luz se expande desde los laterales.

Crear una secuencia

  1. Configure el patrón, velocidad, duración y color del Paso 1.
  2. Presione + Agregar paso para incorporar otro efecto.
  3. Repita el proceso hasta completar la secuencia deseada.
  4. Presione Aplicar secuencia avanzada.
Los cambios no se aplican hasta presionar Aplicar secuencia avanzada.

Ejemplo de secuencia

Paso Patrón Velocidad Duración Color
1 Recorrido Muy rápido 6 s Naranja
2 Recorrido Muy rápido 6 s Blanco
3 Expansión desde lados Normal 8 s Naranja

6 Recomendaciones de uso

Antes de utilizar

  • Verifique la carga de batería antes de comenzar una jornada.
  • Asegúrese de que la baliza quede correctamente montada sobre el cono.
  • Mantenga limpia la zona de iluminación LED.

Durante el uso

  • Reduzca el brillo cuando necesite mayor autonomía.
  • Utilice el modo rápido para cambios simples de señalización.
  • Utilice el modo avanzado para crear efectos personalizados.
No cargue la baliza si el conector está mojado o sucio. Utilice siempre cables y cargadores USB en buen estado.
